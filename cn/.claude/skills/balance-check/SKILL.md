当此技能被调用时：

1. **从参数中识别平衡领域**。

2. **从 `assets/data/` 和 `design/balance/` 读取相关数据文件**。

3. **从 `design/gdd/` 读取被检查系统的设计文档**。

4. **执行分析**：

   对于**战斗平衡**：
   - 计算每个功率层级的所有武器/技能的 DPS
   - 检查每个层级的击杀时间
   - 识别任何支配其他所有选项的选项（严格更好）
   - 检查防御选项是否可以创建无法击杀的状态
   - 验证伤害类型/抵抗交互是否平衡

   对于**经济平衡**：
   - 映射所有资源来源和消耗及其流速
   - 预测随时间的资源积累
   - 检查无限资源循环
   - 验证金币消耗与金币生成成比例
   - 检查是否有任何物品永远不值得购买

   对于**进度平衡**：
   - 绘制 XP 曲线和力量曲线
   - 检查死区（太长时间没有有意义的进度）
   - 检查力量峰值（能力突然跳跃）
   - 验证内容门与预期玩家力量一致
   - 检查跳过/刷刷策略是否破坏了预期节奏

   对于**战利品平衡**：
   - 计算获取每个稀有度层级的预期时间
   - 检查保底计时器数学
   - 验证没有战利品在任何阶段完全无用
   - 检查库存压力与获取率

5. **输出分析**：

```
## 平衡检查：[系统名称]

### 分析的数据源
- [读取的文件列表]

### 健康摘要：[健康 / 关注 / 严重问题]

### 检测到的异常值
| 项目/值 | 预期范围 | 实际值 | 问题 |
|-----------|---------------|--------|-------|

### 发现的堕落策略
```
