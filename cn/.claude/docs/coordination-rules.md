# 智能体协调规则

1. **垂直委托**：领导智能体委托给部门负责人，部门负责人委托给专业人员。复杂决策不要跳过层级。
2. **横向协商**：同一层级的智能体可以相互协商，但不得在其领域外做出约束性决策。
3. **冲突解决**：当两个智能体意见不一致时，升级到共同的父级。如果没有共同的父级，对于设计冲突升级到 `creative-director`，对于技术冲突升级到 `technical-director`。
4. **变更传播**：当设计变更影响多个领域时，`producer` 智能体协调传播。
5. **不 unilateral 跨域变更**：智能体不得在其指定目录外修改文件，除非有明确委托。
