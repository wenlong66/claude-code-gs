# Unreal Engine 5.7 — PCG（程序内容生成）

**最后验证：** 2026-02-13
**状态：** 生产就绪（UE 5.7 起）
**插件：** `PCG`（内置，在插件中启用）

---

## 概述

**程序内容生成（PCG）** 是 Unreal 的基于节点的框架，用于大规模生成程序内容。旨在为大型开放世界填充植被、岩石、道具、建筑和其他环境细节。

**使用 PCG 用于：**
- 程序植被放置（树木、草、岩石）
- 基于生物群落的环境生成
- 道路/路径生成
- 建筑/结构放置
- 世界细节填充（道具、杂物）

**不要使用 PCG 用于：**
- 游戏逻辑（使用 Blueprints/C++）
- 一次性手动放置（使用编辑器工具）

**⚠️ 注意：** PCG 在 UE 5.0-5.6 中是实验性的，在 UE 5.7 中成为生产就绪。

---

## 核心概念

### 1. **PCG 图**
- 基于节点的图（类似于材质编辑器）
- 定义生成规则

### 2. **PCG 组件**
- 放置在关卡中，执行 PCG 图
- 在定义的空间中生成内容

### 3. **PCG 数据**
- 点数据（位置、旋转、缩放）
- 样条数据（路径、道路、河流）
- 体积数据（密度、生物群落遮罩）

### 4. **节点**
- **采样器**：生成点（Grid、Poisson、Surface）
- **过滤器**：基于规则移除点（Density、Tag、Bounds）
- **修饰符**：变换点（Offset、Rotate、Scale）
- **生成器**：在点处实例化网格/角色
