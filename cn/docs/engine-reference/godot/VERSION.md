## 知识差距警告

LLM 的训练数据可能覆盖 Godot 直至约 4.3。版本 4.4、4.5 和 4.6 引入了模型不知道的重大变更。在建议 Godot API 调用之前务必交叉参考此目录。

## 截止后版本时间线

| 版本 | 发布 | 风险级别 | 关键主题 |
|---------|---------|------------|-----------|
| 4.4 | 约 2025 年中期 | 中等 | Jolt 物理选项、FileAccess 返回类型、着色器纹理类型变更 |
| 4.5 | 约 2025 年晚期 | 高 | 可访问性（AccessKit）、可变参数、@abstract、着色器 baker、SMAA |
| 4.6 | 2026 年 1 月 | 高 | Jolt 默认、glow 重做、Windows 上 D3D12 默认、IK 恢复 |
